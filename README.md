## 📂 Current Project: File Organizer

`organizador_de_archivos.py` is a Python script that automatically organizes files into folders according to their extensions.

The project was developed and tested on Android using **Pydroid 3**, but it can also be adapted to Windows, Linux and macOS by changing the target folder path.

---

## ✨ Features

- Detects each file's extension automatically.
- Creates category folders when they do not already exist.
- Moves files into the appropriate category.
- Prevents existing files from being overwritten.
- Renames duplicate files using a numerical suffix.
- Ignores folders and processes files only.
- Places unsupported extensions in an `Others` folder.
